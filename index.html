Iteration 1. Inline VS Block elements

Write a paragraph containing some span or em tags of different colors. Spans are inline elements, meaning that their default display property value is inline. Those elements will follow the horizontal flow of a document. Inline elements go next to another inline element so you won't have any line break until you decide do insert one (like here with a br tag).

Images are inline element too! That's a simple but super important point to remember ! Feel free to dif deeper with the MDN doc.
